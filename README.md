# Nómada — Turismo sostenible (SPA)

**Nómada** es una Single Page Application (SPA) creada con **HTML, CSS y JavaScript puro** para mostrar experiencias de turismo sostenible, calcular la huella estimada de un viaje y ofrecer planes y recomendaciones responsables.

---
## Características
- Catálogo de experiencias (datos en JavaScript).
- Calculadora de huella/impacto que estima emisiones/consumo según parámetros (km, tipo de transporte, ocupación, noches, etc.).
- Sección de planes y/ o paquetes.
- Componentes UI: cards, tags, botones, modal/overlay.
- Responsive (diseño adaptado a móviles y escritorio).
- Todo implementado con Vanilla JS (sin frameworks).

---

## Tecnologías
- HTML5
- CSS (estilos incluidos inline / en `<style>` dentro de `index.html`)
- JavaScript (Vanilla)
